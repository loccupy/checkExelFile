    Перед использованием скрипта рекомендуется установить самую актуальную версию питона (при разработке
использовался 3.13), для установки библиотек запустить батник "install_requirements.bat".

    Как использовать:
1. В папку <exel_file> поместить файл формата ".xlsx", можно добавлять несколько файлов;
2. Запустить батник "start_checking";
3. Закрыть командную строку. (в перспективе можно настроить закрытие при завершении проверки, оставил
пока для обкатки программы)

    Примечания:
При написании программы был предоставлен один файл поверки, исходя из этого предполагаю, что структура
подобных файлов одинаковая. Программа работает для всех типов. (19 мая 2025)
